PASOS PARA OBTENER TU API KEY DE GEMINI (GRATIS):

1. Ingresa a la página oficial: https://aistudio.google.com/
2. Inicia sesión con cualquier cuenta de Google (Gmail).
3. En el menú de la izquierda, haz clic en el botón azul "Get API Key".
4. Haz clic en el botón "Create API key in new project".
5. Copia el código que aparece (es una serie larga de letras y números).
6. Pega ese código en tu archivo 'config.php' donde dice: $gemini_api_key = 'TU_CODIGO_AQUI';

¡Eso es todo! Con eso, tu sistema Fieldata ya tendrá "cerebro".
